# Solucion.ar

Esta es una aplicacion para el frontend de la aplicacion Solucion.ar en TypeScript


## Installation

Abrir la terminal y correr el siguiente comando

> npm install


## Correr la aplicacion

Abrir la terminal y correr el siguiente comando

> npm run dev


## Eslint

Este repositorio usa Eslint para mantener el codigo limpio y organizado

Se necesita usar la extension Eslint en Visual Studio Code y utilizar el archivo de configuracion dado

Podes encontrar al extension [aqui](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)

[Mas info sobre Eslint](https://eslint.org/)
